# Gesture Pilot WS Relay (Render)

## Deploy (Render Free)
1. Create a new **Web Service** in Render.
2. Connect your GitHub repo.
3. **Root Directory**: `ws-relay`
4. **Build Command**: `npm install`
5. **Start Command**: `npm start`
6. Choose the **Free** plan and deploy.

## Local dev
```bash
cd ws-relay
npm install
npm start
```
Server runs on `http://localhost:3000` and WebSocket on the same host.
